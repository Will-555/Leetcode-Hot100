# 难度：困难

给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

**示例 1：**
输入：`s = "(()"`
输出：`2`
解释：最长有效括号子串是 `"()"`

**示例 2：**
输入：`s = ")()())"`
输出：`4`
解释：最长有效括号子串是 `"()()"`

**示例 3：**
输入：`s = ""`
输出：`0`

**提示：**
- `0 <= s.length <= 3 * 10^4`
- `s[i]` 为 `'('` 或 `')'`

### 题目分析

本题可以通过动态规划或栈来解决，这里介绍**动态规划**解法。

#### 1. 状态定义
- `dp[i]`：以 `s[i]` 结尾的最长有效括号的长度。

#### 2. 状态转移
我们只在 `s[i] == ')'` 时进行转移，因为以 `'('` 结尾的有效括号长度必然是 0。
- **情况一**：`s[i-1] == '('` (形式如 `...()`)
  - `dp[i] = dp[i-2] + 2`
- **情况二**：`s[i-1] == ')'` (形式如 `...))`)
  - 如果匹配到 `s[i - dp[i-1] - 1] == '('`，说明这最后两个 `)` 分别有对应的 `(`。
  - `dp[i] = dp[i-1] + 2 + dp[i - dp[i-1] - 2]`

#### 3. 复杂度
- **时间复杂度**：$O(n)$，遍历一次字符串。
- **空间复杂度**：$O(n)$，需要一个 `dp` 数组。

### 代码实现

```Java
class Solution {
    public int longestValidParentheses(String s) {
        int maxAns = 0;
        int n = s.length();
        int[] dp = new int[n];
        for (int i = 1; i < n; i++) {
            if (s.charAt(i) == ')') {
                if (s.charAt(i - 1) == '(') {
                    dp[i] = (i >= 2 ? dp[i - 2] : 0) + 2;
                } else if (i - dp[i - 1] > 0 && s.charAt(i - dp[i - 1] - 1) == '(') {
                    dp[i] = dp[i - 1] + ((i - dp[i - 1]) >= 2 ? dp[i - dp[i - 1] - 2] : 0) + 2;
                }
                maxAns = Math.max(maxAns, dp[i]);
            }
        }
        return maxAns;
    }
}
```
